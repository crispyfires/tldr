# countryfetch

> Display information about countries in the terminal.
> More information: <https://github.com/nik-rev/countryfetch>.

- Display information about your country:

`countryfetch`

- Display information about a specific country:

`countryfetch {{country_name}}`

- List all countries:

`countryfetch --list-countries`

- Display information about all countries:

`countryfetch --all-countries`
